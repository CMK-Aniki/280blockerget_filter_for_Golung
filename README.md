#280blockerget_filter_for_Golung
280blocker様の当月のブロックルールを取得するためのGo言語で書かれたコードです。
Chrome拡張機能:ublockOriginにインポートできるようAdblockPlus形式ファイル準拠で取得しています。

プログラミング・コードがわからないなど不慣れな方は、添付のexeアプリをご利用下さい。

このコードは改変・再配布等自由です。
